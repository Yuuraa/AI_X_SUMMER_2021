### 실습 구성
- 기기 소개 - Raspberry Pi & NUC & Pi Camera
- 리눅스 Bash 명령어 & Vim
- SSH 상호 연결
- Kafka + Python을 사용한 데이터 전송, 스트리밍
- 컴퓨터 비전 애플리케이션: 사진, 유튜브 영상, Raspberry Pi 영상
  - 얼굴 특징점 추출, 얼굴 인식, 스타일 변환, 객체 검출!
